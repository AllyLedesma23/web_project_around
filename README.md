# Tripleten web_project_around

La página web "Alrededor de los EE.UU" es el séptimo proyecto en el programa de Desarrollo web de TripleTen.
Fue creado utilizando HTML, CSS y poniendo en práctica el nuevo conocimiento de JS para desarrollar una página web
donde los usuarios pueden interactuar añadiendo, eliminando y dando me gusta a las fotos

## Características

- HTML5 Semántico
- Metodología BEM
- Diseño Responsivo
- FIGMA
- JavaScript (conocimientos básicos)
